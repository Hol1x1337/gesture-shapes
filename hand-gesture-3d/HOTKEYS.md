# 🎮 Горячие клавиши Hand Gesture 3D Control v3.0

## Основные команды

| Клавиша | Действие | Описание |
|---------|----------|----------|
| **F11** | Полноэкранный режим | Включить/выключить полноэкранный режим |
| **Esc** | Выход | Выйти из полноэкранного режима или режима презентации |
| **Пробел** | Камера | Запустить/остановить отслеживание рук |
| **R** | Сброс | Вернуть 3D объект в исходное положение |
| **P** | Презентация | Включить/выключить режим презентации |

## Управление вращением объекта

| Клавиша | Действие | Описание |
|---------|----------|----------|
| **←** (стрелка влево) | Вращение влево | Повернуть объект вокруг оси Y |
| **→** (стрелка вправо) | Вращение вправо | Повернуть объект вокруг оси Y |
| **↑** (стрелка вверх) | Вращение вверх | Повернуть объект вокруг оси X |
| **↓** (стрелка вниз) | Вращение вниз | Повернуть объект вокруг оси X |

## Масштабирование

| Клавиша | Действие | Описание |
|---------|----------|----------|
| **+** | Увеличить | Увеличить масштаб объекта |
| **-** | Уменьшить | Уменьшить масштаб объекта |

---

## 📺 Режимы работы

### Обычный режим
- Видны все панели управления
- Доступны все кнопки и элементы интерфейса
- Стандартный размер окна

### Полноэкранный режим (F11)
- Приложение разворачивается на весь экран
- Все элементы управления остаются видимыми
- Нажмите **F11** или **Esc** для выхода

### Режим презентации (P)
- Полноэкранный режим без панелей управления
- Видим только 3D объект и камера
- Идеально для демонстрации жестов
- Нажмите **P** или **Esc** для выхода

---

## 💡 Советы по использованию

1. **Быстрый старт**: Нажмите **Пробел** для быстрого запуска камеры
2. **Демонстрация**: Используйте режим презентации (**P**) для чистого показа
3. **Точная настройка**: Стрелки позволяют точно контролировать вращение
4. **Масштаб**: +/- помогают рассмотреть детали объекта
5. **Сброс**: **R** быстро возвращает объект в начальную позицию

---

## 🖱️ Кнопки интерфейса

- **▶** - Запуск камеры (альтернатива Пробелу)
- **⏹** - Остановка камеры
- **↺** - Сброс объекта (альтернатива R)
- **⛶** - Полноэкранный режим (альтернатива F11)
- **📺** - Режим презентации (альтернатива P)
