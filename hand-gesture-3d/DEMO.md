## Примеры использования

### 1️⃣ Базовое управление позицией

**Действие:** Двигайте ладонью влево-вправо, вверх-вниз

**Результат:**
- Ладонь слева → объект смещается влево
- Ладонь справа → объект смещается вправо
- Ладонь выше → объект поднимается
- Ладонь ниже → объект опускается

**Математика:**
```python
palm_x = 0.3 (слева от центра)
→ position_x = (0.3 - 0.5) * 2 * 3.0 = -1.2

palm_y = 0.7 (ниже центра)
→ position_y = (0.5 - 0.7) * 2 * 2.0 = -0.8
```

---

### 2️⃣ Управление масштабом через Pinch

**Действие:** Сводите и разводите большой и указательный пальцы

**Результат:**
- Пальцы вместе (pinch ≈ 0) → объект маленький (scale ≈ 0.5)
- Пальцы далеко (pinch ≈ 1) → объект большой (scale ≈ 2.0)

**Примеры:**
```python
# Пальцы касаются друг друга
distance = 0.02
pinch = (0.02 - 0.02) / 0.15 = 0.0
scale = 0.5 + 0.0 * 1.5 = 0.5  # Минимальный размер

# Пальцы на среднем расстоянии
distance = 0.095
pinch = (0.095 - 0.02) / 0.15 = 0.5
scale = 0.5 + 0.5 * 1.5 = 1.25  # Средний размер

# Пальцы максимально разведены
distance = 0.17
pinch = (0.17 - 0.02) / 0.15 = 1.0
scale = 0.5 + 1.0 * 1.5 = 2.0  # Максимальный размер
```

---

### 3️⃣ Автоматическое вращение

**Действие:** Двигайте рукой по кругу

**Результат:**
- Горизонтальное движение → вращение вокруг Y оси
- Вертикальное движение → вращение вокруг X оси

**Пример:**
```python
# Движение вправо
palm_x_velocity = +0.1
rotation_y += 0.1 * 2.0 = +0.2 радиан

# Движение вверх
palm_y_velocity = -0.15
rotation_x += -0.15 * 2.0 = -0.3 радиан
```

---

### 4️⃣ Комбинированные жесты

**Сценарий:** Переместить объект вправо и увеличить

**Последовательность:**
1. Двиньте ладонь вправо → объект движется вправо
2. Разведите пальцы → объект увеличивается
3. Объект переместился вправо и стал больше!

**Данные жестов:**
```python
{
    'hand_detected': True,
    'pinch_distance': 0.8,      # Пальцы разведены
    'palm_x': 0.6,              # Справа от центра
    'palm_y': 0.4               # Чуть выше центра
}

# Результат трансформаций:
position_x = (0.6 - 0.5) * 2 * 3.0 = +0.6
position_y = (0.5 - 0.4) * 2 * 2.0 = +0.4
scale = 0.5 + 0.8 * 1.5 = 1.7
```

## Визуализация Landmarks

MediaPipe обнаруживает 21 точку руки:

```
        (0) Wrist
          |
    (1)---(2)---(3)---(4)  Thumb
          |
    (5)---(6)---(7)---(8)  Index
          |
    (9)---(10)--(11)--(12) Middle
          |
    (13)--(14)--(15)--(16) Ring
          |
    (17)--(18)--(19)--(20) Pinky
```

**Ключевые точки для жестов:**
- **Landmark 4**: Кончик большого пальца → для pinch
- **Landmark 8**: Кончик указательного пальца → для pinch
- **Landmark 9**: Центр ладони (MCP среднего пальца) → для позиции

## Советы по использованию

### ✅ Хорошие практики

1. **Освещение**: Используйте равномерное освещение без резких теней
2. **Фон**: Однотонный фон улучшает обнаружение руки
3. **Расстояние**: Держите руку на расстоянии 30-60 см от камеры
4. **Угол**: Камера должна видеть руку спереди или под небольшим углом
5. **Скорость**: Избегайте слишком быстрых движений

### ❌ Чего избегать

1. **Быстрые движения**: MediaPipe может потерять отслеживание
2. **Частичная видимость**: Рука должна быть полностью в кадре
3. **Перекрытие**: Не закрывайте руку другой рукой или объектами
4. **Экстремальные углы**: Избегайте видов сбоку или сзади
5. **Плохой свет**: Избегайте контрового света и теней на руке

## Расширенные техники

### Точное позиционирование

Для точного размещения объекта:
1. Медленно двигайте ладонью к цели
2. Используйте небольшие движения запястьем
3. При необходимости сбросьте позицию кнопкой "Reset"

### Контроль масштаба

Для плавного изменения размера:
1. Начните с разомкнутых пальцев (большой объект)
2. Постепенно сводите пальцы для уменьшения
3. Зафиксируйте размер когда достигнете нужного

### Комбинации жестов

**Пример 1: Вращение вокруг объекта**
- Двигайте рукой по кругу → объект вращается
- Одновременно меняйте pinch для масштабирования

**Пример 2: "Летающий" объект**
- Быстрые движения рукой → быстрое перемещение
- Резкие изменения pinch → пульсация размера

**Пример 3: Точная настройка**
- Медленные движения → плавное перемещение
- Минимальные изменения pinch → тонкая настройка масштаба
