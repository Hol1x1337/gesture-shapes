# Hand Gesture 3D Control - Быстрый старт

## 🚀 Запуск приложения

### 1. Установка зависимостей

```bash
cd hand-gesture-3d
pip install -r requirements.txt
```

### 2. Запуск

```bash
cd src
python main.py
```

## 🎮 Основные жесты

### Одна рука:

| Жест | Действие | Эффект |
|------|----------|--------|
| Движение ладони | Вращение объекта | 🟡 Желтые кольца |

## ✨ Новые функции v2.0

### 1. Выбор типа объекта

В панели управления используйте выпадающий список **"🎲 Объект"**:
- **Куб** - классический куб с цветными гранями
- **Сфера** - гладкая сфера
- **Пирамида** - четырехугольная пирамида
- **Тор** - форма пончика

### 2. Визуализация скелета руки

Когда камера активна, в 3D окне отображаются:
- **Синий скелет** (слева) - ваша правая рука
- **Оранжевый скелет** (справа) - ваша левая рука
- Точки разных цветов показывают суставы и кончики пальцев

### 3. Эффекты обратной связи

При выполнении жестов появляются красивые эффекты:
- Пульсирующие кольца вокруг объекта
- Разлетающиеся частицы
- Цвет зависит от типа жеста

### 4. HD качество камеры

Камера теперь работает в разрешении **1280×720** для лучшей детализации.

## 💡 Советы

1. **Освещение**: Обеспечьте хорошее освещение для точного распознавания
2. **Расстояние**: Держите руку на расстоянии 30-60 см от камеры
3. **Фон**: Используйте однотонный фон для лучшей точности
4. **Движения**: Делайте плавные движения для лучшего контроля

## 🎯 Примеры использования

### Вращение объекта:
1. Поднимите одну руку перед камерой
2. Двигайте ладонью влево-вправо или вверх-вниз
3. Объект будет вращаться соответственно
4. Появятся желтые кольца эффекта

### Изменение размера:
1. Используйте большой и указательный пальцы
2. Сводите их вместе - объект уменьшается
3. Разводите их - объект увеличивается
4. Появятся зеленые кольца эффекта

### Приближение/отдаление:
1. Соедините большой и указательный пальцы (жест OK)
2. Приближайте руку к камере - объект приближается
3. Отдаляйте руку от камеры - объект отдаляется
4. Появятся голубые кольца эффекта

### Смена объекта:
1. В панели управления найдите "🎲 Объект"
2. Выберите из списка: Куб, Сфера, Пирамида или Тор
3. Объект мгновенно изменится

## 🔍 Troubleshooting

### Камера не открывается
- Проверьте доступ к камере в настройках ОС
- Убедитесь, что камера не используется другим приложением
- Попробуйте изменить `camera_id` в коде (0, 1, 2...)

### Низкая производительность
- Закройте другие приложения, использующие камеру
- Уменьшите окно приложения
- Проверьте нагрузку на CPU в диспетчере задач

### Рука не обнаруживается
- Улучшите освещение
- Держите руку ближе к камере (30-60 см)
- Избегайте быстрых движений
- Убедитесь, что рука полностью видна в кадре

### Скелет руки не отображается
- Убедитесь, что камера активна (статус "АКТИВЕН")
- Покажите руку четко в центре кадра
- Проверьте, что MediaPipe правильно установлен
