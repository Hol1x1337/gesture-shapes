# 🎉 Обновление версии 1.1.0 - Поддержка двух рук и улучшенный интерфейс

## ✨ Что нового в версии 1.1.0

### 🖐️ Поддержка двух рук

Теперь приложение может одновременно отслеживать **до 2 рук**:

- **Правая рука** (синий цвет landmarks) - основное управление объектом
  - Перемещение объекта по X/Y
  - Управление масштабом через pinch
  - Автоматическое вращение при движении

- **Левая рука** (зеленый цвет landmarks) - дополнительное управление
  - Резервное управление
  - Возможность расширения функциональности

### 📺 Увеличенные окна

- **Размер окна приложения:** 1600x1000 пикселей (было ~1200x800)
- **Минимальный размер:** 1400x900 пикселей
- **Разрешение камеры:** 640x480 (было 320x240)
- **Улучшенное качество видео** для лучшей детализации

### 🔧 Технические улучшения

#### HandTracker
- `max_num_hands = 2` - поддержка двух рук
- Разрешение камеры: 640x480 (в 4 раза больше пикселей)
- Данные рук хранятся в списке `hands_data`
- Каждая рука имеет поле `handedness` ('Left' или 'Right')

#### GLWidget
- Отдельные переменные для левой и правой руки
- Правая рука управляет основной трансформацией
- Левая рука готова для расширенной логики
- Улучшенное освещение с двумя источниками света

#### CameraWidget
- Метод `_draw_overlay()` для рисования информации
- Отображение данных для каждой руки отдельно
- Полупрозрачный фон под текстом
- Адаптивное позиционирование текста

#### MainWindow
- Splitter с пропорциями 55%/45% (камера/3D)
- Увеличенные отступы (15px)
- Современные стили через QSS (Qt Style Sheets)
- Динамическое обновление статуса

## 🎮 Как использовать две руки

### Базовое управление (одна правая рука)

1. Покажите **правую руку** камере
2. Двигайте ладонью для перемещения объекта
3. Сводите пальцы для изменения размера
4. Объект автоматически вращается при движении

### Расширенное управление (две руки)

1. Покажите **обе руки** камере
2. **Правая рука** управляет объектом (как раньше)
3. **Левая рука** отображается синим цветом
4. В будущем левая рука сможет:
   - Управлять цветом объекта
   - Переключать режимы
   - Контролировать дополнительные параметры

### Индикация в интерфейсе

**Видео с камеры:**
- Зеленые landmarks → Левая рука
- Синие landmarks → Правая рука
- Текст overlay показывает данные обеих рук

**Панель информации:**
```
🔵 ОСНОВНАЯ - RIGHT РУКА #1
  Pinch: 0.45 ○
  Позиция X: +0.23
  Позиция Y: -0.15

🟢 ДОПОЛНИТЕЛЬНАЯ - LEFT РУКА #2
  Pinch: 0.78 ●
  Позиция X: -0.34
  Позиция Y: +0.12

✅ Обе руки обнаружены!
```

## 💡 Советы по использованию двух рук

### Оптимальное расположение
- Держите обе руки на расстоянии 30-60 см от камеры
- Расположите руки на ширине плеч
- Избегайте перекрытия рук друг другом

### Лучшие практики
1. **Основное управление:** Используйте правую руку
2. **Дополнительные жесты:** Экспериментируйте с левой рукой
3. **Pinch обеими руками:** Можно контролировать два параметра одновременно
4. **Быстрое переключение:** Убирайте одну руку для упрощения управления

### Troubleshooting
- **Одна рука не обнаруживается:** Улучшите освещение, проверьте чтобы обе руки были полностью видны
- **Руки путаются:** MediaPipe автоматически определяет левую/правую руку по ориентации
- **Низкий FPS с двумя руками:** Это нормально, обработка двух рук требует больше ресурсов
