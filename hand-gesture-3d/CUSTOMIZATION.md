#  Расширенная кастомизация 3D объектов

## Обзор

Hand Gesture 3D Control v3.0 поддерживает расширенную кастомизацию 3D объектов через **отдельное диалоговое окно настроек**, которое не загромождает основной интерфейс.

---

## 🎯 Как открыть настройки

### Способ 1: Кнопка в интерфейсе
Нажмите кнопку **** в нижней панели управления (рядом с кнопками полноэкранного режима и презентации). Откроется модальное окно с полным набором настроек.

### Способ 2: Горячая клавиша (планируется)
В будущих версиях будет добавлена горячая клавиша для быстрого доступа.

---

## ⚙️ Доступные настройки в диалоге

### 1. Размер и материал 📐

#### **Базовый размер объекта**
- **Слайдер**: 0.10 - 1.00
- **По умолчанию**: 0.30
- Отображается текущее значение рядом со слайдером
- Применяется ко всем типам объектов

#### **Блеск материала (Shininess)**
- **Слайдер**: 0 - 128
- **По умолчанию**: 80
- Отображается текущее значение
- Контролирует интенсивность бликов

---

### 2. Автовращение 🔄

#### **Кнопка включения/выключения**
- Переключает автоматическое вращение объекта вокруг оси Y
- Кнопка меняет цвет: зеленая (выключено) → красная (включено)

#### **Скорость вращения**
- **Слайдер**: 0.1 - 5.0
- **По умолчанию**: 0.5
- Отображается текущее значение

---

### 3. Режимы отображения 

Три чекбокса для переключения режимов:
- ✓ **Каркасный режим (Wireframe)** - показывает только ребра
- ✓ **Показывать сетку** - сетка пола (включено по умолчанию)
- ✓ **Показывать оси координат** - оси X/Y/Z

---

### 4. Цвета объекта 🎨

#### **Цвета граней куба**
Шесть цветных кнопок для каждой грани:
- Передняя (синий)
- Задняя (красный)
- Верхняя (зеленый)
- Нижняя (желтый)
- Левая (голубой)
- Правая (пурпурный)

**Как изменить:**
1. Нажмите на кнопку нужной грани
2. Выберите цвет в диалоге QColorDialog
3. Цвет мгновенно применится, кнопка обновится

#### **Основной цвет для других объектов**
Кнопка "Изменить основной цвет" для сферы, пирамиды, тора, конуса, цилиндра.
Создает градиент с автоматическим вторичным цветом.

---

### 5. Сброс настроек ↺

Красная кнопка **"Сбросить все настройки"** возвращает все параметры к значениям по умолчанию.

## 💡 Преимущества нового подхода

### ✅ Чистый основной интерфейс
- 3D вид занимает максимум пространства
- Нижняя панель компактная и минималистичная
- Нет перегруженности элементами управления

### ✅ Модальное окно настроек
- Все настройки в одном месте
- Не отвлекает от основной работы
- Легко закрыть когда не нужно
- Сохраняет состояние при закрытии

### ✅ Удобство использования
- Мгновенное применение изменений
- Визуальная обратная связь (значения слайдеров)
- Цветные кнопки для быстрого выбора
- Понятные иконки и подписи

## 🔧 Программное API

Для разработчиков доступны методы в GLWidget:

```python
# Изменение цвета грани куба
gl_widget.set_cube_color('front', [1.0, 0.0, 0.0, 1.0])

# Изменение основного цвета
gl_widget.set_object_color([0.5, 0.5, 1.0, 1.0])

# Настройка материала
gl_widget.set_material_properties(shininess=100.0)

# Переключение режимов
gl_widget.toggle_wireframe()
gl_widget.toggle_grid()
gl_widget.toggle_axes()

# Автовращение
gl_widget.toggle_auto_rotate()
gl_widget.set_auto_rotate_speed(2.0)

# Размер
gl_widget.set_base_size(0.5)

# Сброс
gl_widget.reset_customization()
```
