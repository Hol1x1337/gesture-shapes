### ✅ Индикаторы осей координат

**Три цветные оси из центра:**
- **Ось X** - красная (вправо)
- **Ось Y** - зеленая (вверх)
- **Ось Z** - синяя (вперед)

**Характеристики:**
- Длина каждой оси: 2.0 единицы
- Маркеры на концах осей (квадраты)
- Отрисовка без освещения (яркие цвета)
- Всегда видны независимо от вращения

**Визуализация:**
```
        Y (зеленая)
        ↑
        |
        |
        |______→ X (красная)
       /
      /
     ↓ Z (синяя)
```

**Назначение:**
- Понимание ориентации объекта в пространстве
- Контроль направления вращения
- Профессиональная 3D визуализация

---

### ✅ Сетка пола для понимания пространства

**Детали сетки:**
- Размер: 10x10 единиц
- divisions: 20 линий по каждой оси
- Шаг между линиями: 0.5 единиц
- Цвет: полупрозрачный серый (30% opacity)
- Расположение: на уровне Y = -5.0 (под объектом)

**Эффект:**
- Создает ощущение глубины и перспективы
- Помогает оценить расстояние до объекта
- Профессиональный вид как в CAD системах

**Визуализация:**
```
    ┌──┬──┬──┬──┬──┐
    │  │  │  │  │  │
    ├──┼──┼──┼──┼──┤
    │  │  │  │  │  │  ← Сетка пола
    ├──┼──┼──┼──┼──┤
    │  │  │🎲│  │  │  ← Куб в центре
    ├──┼──┼──┼──┼──┤
    │  │  │  │  │  │
    └──┴──┴──┴──┴──┘
```

## 🔘 Кнопки управления - баланс и логика

### ✅ Кнопки одинаковой ширины

**Изменения:**
- Все кнопки: `min-width: 140px`
- Одинаковая высота: `min-height: 45px`
- Единый padding: `14px 28px`
- Border-radius: 10px

**Расположение:**
```
┌─────────────────────────────┐
│ [▶ Запустить] [⏹ Стоп]     │ ← Одна строка
│                             │
│ [↺ Сбросить объект]         │ ← Отдельная строка
└─────────────────────────────┘
```

**Преимущества:**
- Визуальный баланс
- Профессиональный вид
- Удобное нажатие (большие кнопки)

---

### ✅ Логическая группировка

**Группа 1: Управление камерой**
- ▶ Запустить камеру
- ⏹ Остановить
- Расположены горизонтально рядом

**Группа 2: Управление объектом**
- ↺ Сбросить объект
- На отдельной строке
- Выделена фиолетовым цветом

**Цветовое кодирование:**
- Зеленый → Запуск (позитивное действие)
- Красный → Остановка (предупреждение)
- Фиолетовый → Сброс (нейтральное)

## 👋 Информация о жестах - читаемость и иконки

### ✅ Крупный текст и иконки


### ✅ Отдельные иконки для жестов

**Визуальная система:**
- 👌 - жест OK (zoom)
- ✋ - открытая ладонь (стоп)
- 🖐️ - движение руки (вращение)
- 🔵 - правая рука
- 🟢 - левая рука
- 🔄 - активное вращение
- ⏸️ - мертвая зона

**Преимущества:**
- Мгновенное распознавание
- Не нужно читать текст
- Интуитивно понятно

## 🟢 Статус "АКТИВЕН" - максимальная заметность

- **Зеленый градиентный фон** (rgba(16, 185, 129, 0.15))

- Серая рамка
- Менее яркий текст

**Контраст:**
- Активный статус сразу бросается в глаза
- Невозможно пропустить изменение состояния
- Цветовая дифференциация состояний

## 🎯 Как использовать новый интерфейс

### При первом запуске

1. **Видите placeholder камеры** с иконкой
2. **Нажимаете "▶ Запустить"**
3. **Появляется видеопоток** с:
   - Рамкой зоны обнаружения
   - Счётчиком FPS вверху
   - Зеленой точкой активности
   - Информацией о руках

### Работа с 3D видом

1. **Видите сетку пола** под кубом
2. **Видите цветные оси** из центра
3. **Управляете жестами** как раньше
4. **Следите за FPS** для контроля производительности

### Использование кнопок

1. **Наводите мышь** на кнопку
2. **Читаете тултип** с подсказкой
3. **Нажимаете** с уверенностью
