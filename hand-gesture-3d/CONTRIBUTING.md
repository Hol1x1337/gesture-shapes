# 🤝 Руководство по внесению вклада

Спасибо за интерес к проекту Hand Gesture 3D Control! Мы приветствуем любой вклад в развитие проекта.

## 📋 Как можно помочь

### 1. Сообщить об ошибке

Если вы нашли баг:
- Проверьте, не сообщалось ли об этой ошибке ранее в [Issues](https://github.com/yourusername/hand-gesture-3d/issues)
- Создайте новый issue с подробным описанием проблемы
- Укажите версию Python, операционную систему и шаги для воспроизведения

### 2. Предложить новую функцию

Для предложений новых функций:
- Откройте issue с меткой "enhancement"
- Опишите функциональность и её пользу
- Обсудите реализацию с сообществом

### 3. Исправить код

1. Форкните репозиторий
2. Создайте ветку: `git checkout -b feature/your-feature`
3. Внесите изменения
4. Протестируйте код
5. Отправьте pull request

## 💻 Стандарты кода

### Python код

- Следуйте PEP 8
- Используйте типизацию (type hints)
- Добавляйте docstrings к функциям и классам
- Максимальная длина строки: 100 символов

### Коммиты

Используйте понятные сообщения коммитов:
```
feat: добавлена поддержка нового жеста
fix: исправлена ошибка с FPS
docs: обновлена документация
style: форматирование кода
refactor: рефакторинг модуля жестов
test: добавлены тесты
```

### Документация

- Обновляйте README.md при добавлении новых функций
- Добавляйте комментарии к сложному коду
- Используйте markdown для форматирования

## 🧪 Тестирование

Перед отправкой pull request:
1. Убедитесь, что приложение запускается без ошибок
2. Протестируйте новые функции
3. Проверьте производительность (FPS > 30)
4. Убедитесь, что нет утечек памяти

## 📝 Pull Request процесс

1. **Описание**: Подробно опишите изменения
2. **Связанные issues**: Укажите номера связанных issues
3. **Тестирование**: Подтвердите, что код протестирован
4. **Документация**: Обновите документацию если нужно

## ❓ Вопросы?

Если у вас есть вопросы, создайте issue с меткой "question" или свяжитесь с maintainer'ом.
