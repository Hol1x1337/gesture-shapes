### 👋 3. Расширенные жесты

**Добавлено 4 новых жеста:**

#### 🤛 Жест "Кулак"
- **Как сделать:** Согните все пальцы в кулак
- **Действие:** Пауза вращения (готовится к реализации)
- **Индикация:** Лог в консоли

#### ☝️ Жест "Указательный палец вверх"
- **Как сделать:** Поднимите только указательный палец, остальные согните
- **Действие:** Мгновенный сброс позиции объекта к начальному состоянию
- **Использование:** Быстрый возврат к стандартному виду

#### ✌️ Жест "Победа" (V)
- **Как сделать:** Поднимите указательный и средний пальцы буквой V
- **Действие:** Переключение на следующий 3D объект
- **Использование:** Быстрая смена объекта без меню

#### ❤️ Жест "Сердце"
- **Как сделать:** Большой и указательный вместе, остальные выпрямлены
- **Действие:** Активация специального эффекта (готовится)
- **Индикация:** Лог в консоли

### ⚙️ 4. Система настроек

**Создан централизованный модуль настроек** (`config/settings.py`)

**Категории настроек:**

#### Визуальные настройки
```python
"visual": {
    "theme": "dark",              # Тема оформления
    "show_grid": True,            # Показывать сетку
    "show_axes": False,           # Показывать оси координат
    "shadow_enabled": False,      # Тени (в разработке)
    "reflection_enabled": False,  # Отражения (в разработке)
    "particle_effects": False,    # Эффекты частиц
    "glow_effect": True,          # Свечение объекта
}
```

#### Настройки камеры
```python
"camera": {
    "mode": "fixed",              # Режим камеры (fixed/orbital/free)
    "distance": 9.0,              # Расстояние до объекта
    "fov": 45.0,                  # Угол обзора
    "auto_rotate": False,         # Автоматическое вращение
    "auto_rotate_speed": 0.5,     # Скорость автовращения
}
```

#### Чувствительность жестов
```python
"gestures": {
    "rotation_speed": 3.0,        # Скорость вращения
    "dead_zone": 0.1,             # Мертвая зона
    "pinch_threshold": 0.5,       # Порог pinch жеста
    "scale_sensitivity": 2.0,     # Чувствительность масштаба
    "smoothing_factor": 0.15,     # Фактор сглаживания
}
```

#### Материалы и цвета
```python
"material": {
    "type": "standard",           # Тип материала
    "color_scheme": "blue",       # Цветовая схема
    "custom_color": [0.4, 0.5, 0.95],
    "transparency": 0.0,          # Прозрачность
    "shininess": 80.0,            # Блеск
}
```

**Настройки сохраняются в файле `settings.json` автоматически!**

## 🚀 Как использовать новые функции

### 1. Просмотр статистики производительности
- Запустите приложение
- Смотрите в нижнюю правую панель
- FPS обновляется каждую секунду
- Цвет индикатора показывает качество работы

### 2. Использование новых объектов
- Выберите объект из выпадающего списка ИЛИ
- Покажите жест "Победа" (V) для быстрого переключения

### 3. Расширенные жесты
- **Кулак:** Согните все пальцы → пауза
- **Указательный вверх:** Поднимите один палец → сброс позиции
- **Победа (V):** Два пальца вверх → следующий объект
- **Сердце:** Сложите сердце → спецэффект

### 4. Настройка чувствительности
- Откройте `src/config/settings.py`
- Измените значения в разделе `"gestures"`
- Сохраните файл и перезапустите приложение

1. **Для лучшего FPS:**
   - Закройте другие приложения

   - Делайте четкие движения

   - Используйте жест "Указательный вверх" для быстрого сброса
   - Жест "Победа" экономит время на переключении объектов
   - Следите за FPS - если падает ниже 30, упростите сцену
