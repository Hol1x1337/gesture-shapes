## [1.2.0] - 2026-05-06

### 🎮 Новая интуитивная система управления жестами

#### ✨ Добавлено
- **Жест "OK" для zoom** - соедините большой и указательный пальцы и двигайте руку ближе/дальше от камеры для приближения/отдаления к кубу
- **Управление глубиной (depth)** - вычисление Z координаты руки для контроля zoom
- **Разделение функций для двух рук**:
  - Правая рука → вращение куба
  - Левая рука → масштаб через pinch
- **Визуальная индикация режимов** - overlay показывает текущий режим (ZOOM/ROTATION/STOP)
- **Цветовая кодировка глубины** - зеленый (приближение), желтый (нейтрально), оранжевый (отдаление)
- **Обновленная инструкция** с новыми жестами и режимами

#### 🔧 Изменено
- Полностью переработана логика `update_gesture_data()` в GLWidget
- Добавлен параметр `position_z` для управления глубиной камеры (диапазон [-10, -2])
- Увеличена скорость вращения для одной руки до 3.0 (было 2.0)
- Скорость вращения для двух рук установлена на 2.5
- Диапазон масштаба для левой руки: [0.5, 2.5]
- Порог обнаружения жеста OK: расстояние < 0.05 между кончиками пальцев
- Глубина вычисляется как среднее Z координат запястья, центра ладони и основания указательного пальца

#### 🎯 Режимы работы

**Одна рука:**
- Жест OK + движение вперед/назад → Zoom (приближение/отдаление)
- Простое движение → Вращение куба
- Открытая ладонь → Остановка всех действий

**Две руки:**
- Правая рука → Вращение (независимо от левой)
- Левая рука → Масштаб через pinch (независимо от правой)
- Одновременное управление обоими параметрами
- Открытая ладонь любой руки → Стоп

#### 📊 Технические детали
- Нормализация глубины: диапазон [-0.1, 0.1] → [0.0, 1.0]
- Маппинг zoom: `target_z = -10.0 + depth * 8.0`
- Маппинг масштаба (две руки): `scale = 0.5 + pinch * 2.0`
- Мертвая зона сохраняется: 0.15 для предотвращения случайного вращения
- Интерполяция position_z добавлена для плавного zoom

#### 💡 Преимущества новой системы
- ✅ Интуитивное управление zoom через глубину
- ✅ Разделение функций для двух рук повышает точность
- ✅ Четкая визуальная обратная связь о текущем режиме
- ✅ Профессиональные презентации объекта
- ✅ Независимый контроль вращения и масштаба

## [1.1.1] - 2026-05-06

### 🛑 Исправлено постоянное вращение куба

#### ✨ Добавлено
- **Жест "открытая ладонь"** для остановки вращения - раскройте все пальцы чтобы остановить вращение куба
- **Мертвая зона (dead zone)** с порогом 0.15 - вращение только при значительных движениях руки
- **Визуальная индикация статуса** вращения на видео и в панели информации
- **Цветовые подсказки**: желтый текст при остановке, серый при активности
- **Детальный статус** в информационной панели с объяснением текущего режима

#### 🔧 Изменено
- Вращение теперь происходит только когда:
  1. Ладонь НЕ открыта (пальцы не выпрямлены все)
  2. Движение превышает порог мертвой зоны (0.15)
- Добавлен метод `_detect_open_palm()` в HandTracker для распознавания жеста
- Обновлена логика `update_gesture_data()` в GLWidget с проверкой условий вращения
- Overlay на камере показывает статус: "✋ STOP ROTATION" или "🔄 Rotation active"
- Инструкция обновлена с информацией о новых возможностях контроля

#### 🎯 Преимущества
- ✅ Куб больше не вращается сам по себе
- ✅ Полный контроль над вращением через жесты
- ✅ Защита от случайного вращения при дрожании рук
- ✅ Возможность зафиксировать объект в нужной ориентации
- ✅ Четкая визуальная обратная связь

#### 📊 Технические детали
- Порог мертвой зоны: 0.15 (настраиваемый)
- Требуется 4+ выпрямленных пальца для жеста "открытая ладонь"
- Скорость вращения: 2.0 радиан/единицу движения (настраиваемая)
- Проверка каждого пальца отдельно для точного распознавания
