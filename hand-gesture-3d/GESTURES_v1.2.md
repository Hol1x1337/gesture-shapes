# 🎮 Новая система управления жестами - Версия 1.2.0

## ✨ Что нового

Полностью переработанная интуитивная система управления 3D кубом через жесты рук с поддержкой различных режимов для одной и двух рук.

---

## 🖐️ Управление одной рукой

### Режим 1: Жест "OK" → Zoom (приближение/отдаление) 👌

**Как выполнить:**
1. Соедините кончики **большого** и **указательного** пальцев
2. Остальные пальцы могут быть свободны
3. Двигайте рукой **ближе к камере** или **дальше от камеры**

**Результат:**
- Рука **ближе** к камере → **приближение** к кубу (zoom in)
- Рука **дальше** от камеры → **отдаление** от куба (zoom out)
- Диапазон zoom: от -10.0 до -2.0 по оси Z

**Визуальная индикация:**
```
На видео: 👌 ZOOM MODE (depth: 0.75)
В панели: 👌 ZOOM: ПРИБЛИЖЕНИЕ (глубина: 0.75)
```

**Технические детали:**
- Порог обнаружения жеста OK: расстояние < 0.05 между кончиками пальцев
- Глубина вычисляется как среднее Z координат запястья, центра ладони и основания указательного пальца
- Нормализация глубины: диапазон [-0.1, 0.1] → [0.0, 1.0]
- Маппинг к позиции Z: `target_z = -10.0 + depth * 8.0`

**Примеры использования:**
```
Глубина 0.0 (далеко) → Z = -10.0 (максимальное отдаление)
Глубина 0.5 (средне) → Z = -6.0  (средняя дистанция)
Глубина 1.0 (близко) → Z = -2.0  (максимальное приближение)
```

---

### Режим 2: Простое движение → Вращение куба 🔄

**Как выполнить:**
1. Держите руку в обычном положении (без жеста OK)
2. Двигайте рукой в нужном направлении

**Результат:**
- Вправо → вращение куба вокруг оси Y (против часовой)
- Влево → вращение куба вокруг оси Y (по часовой)
- Вверх → вращение куба вокруг оси X (вперед)
- Вниз → вращение куба вокруг оси X (назад)

**Визуальная индикация:**
```
На видео: 🔄 ROTATION
В панели: 🔄 Вращение активно
```

**Технические детали:**
- Мертвая зона: 0.15 (предотвращает случайное вращение)
- Скорость вращения: 3.0 радиан/единицу движения
- Только значительные движения (>0.15) вызывают вращение

---

### Режим 3: Открытая ладонь → Стоп ✋

**Как выполнить:**
1. Раскройте все 5 пальцев
2. Держите ладонь плоско

**Результат:**
- **Все действия останавливаются**
- Нет вращения
- Нет zoom
- Куб замирает в текущем состоянии

**Визуальная индикация:**
```
На видео: ✋ STOP ALL
В панели: ✋ ВСЕ ДЕЙСТВИЯ ОСТАНОВЛЕНЫ
```

## 🖐️🖐️ Управление двумя руками

Когда обнаружены обе руки, каждая выполняет свою функцию:

### Правая рука (синие landmarks) → Вращение 🔵

**Функция:** Управление вращением куба

**Как использовать:**
- Двигайте правой рукой для вращения
- Мертвая зона: 0.15
- Скорость вращения: 2.5 радиан/единицу движения

**Преимущество:** Точный контроль вращения без влияния на масштаб

---

### Левая рука (зеленые landmarks) → Масштаб 🟢

**Функция:** Управление размером куба через pinch

**Как использовать:**
- Сведите большой и указательный пальцы → куб уменьшается
- Разведите пальцы → куб увеличивается
- Диапазон масштаба: от 0.5 до 2.5

**Визуальная индикация:**
```
Pinch 0.0 → Scale = 0.5 (минимум)
Pinch 0.5 → Scale = 1.5 (средний)
Pinch 1.0 → Scale = 2.5 (максимум)
```

**Преимущество:** Независимое управление масштабом и вращением одновременно

---

## 📊 Сравнение режимов

| Режим | Одна рука | Две руки |
|-------|-----------|----------|
| **Вращение** | Движение рукой | Правая рука двигается |
| **Zoom** | Жест OK + глубина | Не доступно |
| **Масштаб** | Не доступно | Левая рука pinch |
| **Стоп** | Открытая ладонь | Открытая ладонь любой руки |
| **Точность** | Средняя | Высокая (разделение функций) |

## 💡 Советы по использованию

### Для жеста OK

1. **Точное выполнение:** Соедините именно кончики пальцев
2. **Освещение:** Равномерное освещение улучшает распознавание
3. **Расстояние:** Держите руку на расстоянии 30-50 см от камеры
4. **Стабильность:** Медленные движения дают лучший результат

### Для вращения

1. **Значительные движения:** Преодолейте мертвую зону (0.15)
2. **Плавность:** Резкие движения могут вызвать overshoot
3. **Фиксация:** Используйте открытую ладонь для остановки

### Для двух рук

1. **Разделение:** Держите руки на расстоянии друг от друга
2. **Приоритет:** Правая рука главнее для вращения
3. **Координация:** Практикуйте одновременное управление

## 🐛 Troubleshooting

### Жест OK не распознается

**Причины:**
- Пальцы недостаточно близко друг к другу
- Плохое освещение
- Рука слишком далеко от камеры

**Решения:**
1. Сведите кончики пальцев максимально близко
2. Улучшите освещение
3. Приблизьте руку к камере (30-40 см)
4. Проверьте overlay - должно показывать "👌 ZOOM MODE"

### Zoom работает наоборот

**Причина:** Неправильное понимание направления

**Решение:**
- Рука **ближе** к камере = **приближение** к объекту
- Рука **дальше** от камеры = **отдаление** от объекта
- Это интуитивно: как будто вы двигаете камеру

### Вращение слишком быстрое/медленное

**Решение:** Отрегулируйте `rotation_speed` в коде (см. раздел "Настройка параметров")

### Две руки путаются

**Причина:** MediaPipe неправильно определяет левую/правую

**Решение:**
1. Держите руки четко раздельно
2. Не перекрещивайте руки
3. Убедитесь что обе руки полностью видны в кадре
