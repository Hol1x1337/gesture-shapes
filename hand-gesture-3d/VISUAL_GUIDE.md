## Математическая модель жестов

### 1️⃣ Landmarks руки (21 точка)

```
        (0) Wrist
          |
    (1)───(2)───(3)───(4)  ← Thumb (большой палец)
          |
    (5)───(6)───(7)───(8)  ← Index (указательный)
          |
    (9)──(10)──(11)──(12)  ← Middle (средний) ★ Центр ладони
          |
   (13)──(14)──(15)──(16)  ← Ring (безымянный)
          |
   (17)──(18)──(19)──(20)  ← Pinky (мизинец)
```

**Ключевые точки:**
- **Landmark 4**: Кончик большого пальца → для pinch
- **Landmark 8**: Кончик указательного пальца → для pinch
- **Landmark 9**: MCP среднего пальца → центр ладони для позиции

---

### 2️⃣ Вычисление Pinch жеста

```python
# Координаты кончиков пальцев
thumb_tip = landmarks[4]   # [x, y, z]
index_tip = landmarks[8]   # [x, y, z]

# Евклидово расстояние (2D проекция)
distance = √((x₈ - x₄)² + (y₈ - y₄)²)

# Нормализация в диапазон [0, 1]
# Типичные значения:
#   0.02 = пальцы касаются друг друга
#   0.17 = пальцы максимально разведены
pinch = clip((distance - 0.02) / 0.15, 0, 1)

# Примеры:
# distance = 0.02 → pinch = 0.0 (минимум)
# distance = 0.095 → pinch = 0.5 (середина)
# distance = 0.17 → pinch = 1.0 (максимум)
```

**Визуализация:**

```
Пальцы вместе (pinch ≈ 0):       Пальцы далеко (pinch ≈ 1):
     👌                               ✌️
    /                                  \
   O                                    O
```

### 3️⃣ Нормализация позиции ладони

```python
# Центр ладони из MediaPipe (диапазон [0, 1])
palm_x_raw = landmarks[9][0]  # 0 (слева) → 1 (справа)
palm_y_raw = landmarks[9][1]  # 0 (сверху) → 1 (снизу)

# Маппинг в [-1, 1] для OpenGL
normalized_x = (palm_x_raw - 0.5) * 2
normalized_y = (0.5 - palm_y_raw) * 2  # Инвертировано!

# Примеры:
# palm_x_raw = 0.0 → normalized_x = -1.0 (левый край)
# palm_x_raw = 0.5 → normalized_x =  0.0 (центр)
# palm_x_raw = 1.0 → normalized_x = +1.0 (правый край)

# palm_y_raw = 0.0 → normalized_y = +1.0 (верх)
# palm_y_raw = 0.5 → normalized_y =  0.0 (центр)
# palm_y_raw = 1.0 → normalized_y = -1.0 (низ)
```

**Координатная система:**

```
MediaPipe:                   OpenGL:
(0,0) ───── X (1,0)         (-1,+1) ─── X (+1,+1)
  │                            │
  │                              │
  Y                              Y
  │                              │
(0,1)                          (-1,-1) ─── X (+1,-1)
```

### 4️⃣ Маппинг к трансформациям 3D объекта

```python
# === ПОЗИЦИЯ ===
# Усиленная чувствительность для лучшего контроля
position_x = palm_x * 3.0  # Диапазон [-3, +3]
position_y = palm_y * 2.0  # Диапазон [-2, +2]
position_z = -5.0          # Фиксированная глубина

# Примеры:
# palm_x = -1.0 → position_x = -3.0 (лево)
# palm_x =  0.0 → position_x =  0.0 (центр)
# palm_x = +1.0 → position_x = +3.0 (право)

# === МАСШТАБ ===
# Pinch контролирует размер объекта
scale = 0.5 + pinch * 1.5  # Диапазон [0.5, 2.0]

# Примеры:
# pinch = 0.0 → scale = 0.5 (маленький)
# pinch = 0.5 → scale = 1.25 (средний)
# pinch = 1.0 → scale = 2.0 (большой)

# === ВРАЩЕНИЕ ===
# Скорость изменения позиции → угловая скорость
rotation_speed = 2.0  # радиан за единицу движения

target_rotation_y += palm_x * rotation_speed  # Горизонталь → Y ось
target_rotation_x -= palm_y * rotation_speed  # Вертикаль → X ось

# Примеры:
# Движение вправо (palm_x > 0) → вращение против часовой (Y+)
# Движение влево (palm_x < 0) → вращение по часовой (Y-)
# Движение вверх (palm_y < 0) → вращение вперед (X+)
# Движение вниз (palm_y > 0) → вращение назад (X-)
```

### 5️⃣ Плавная интерполяция (Lerp)

```python
# Линейная интерполяция для плавного движения
def lerp(current, target, factor=0.15):
    """
    factor = 0.1 → очень плавно, медленно
    factor = 0.15 → баланс (используется)
    factor = 0.3 → быстро, менее плавно
    factor = 1.0 → мгновенно (без интерполяции)
    """
    return current + (target - current) * factor

# Применение к каждой трансформации:
position_x = lerp(position_x, target_position_x, 0.15)
position_y = lerp(position_y, target_position_y, 0.15)
scale = lerp(scale, target_scale, 0.15)
rotation_x = lerp(rotation_x, target_rotation_x, 0.15)
rotation_y = lerp(rotation_y, target_rotation_y, 0.15)
```

**Визуализация интерполяции:**

```
Целевая позиция:  ████████████░░░░░░░░░░░░

Кадр 1:  ███░░░░░░░░░░░░░░░░░░░░░░░  (current = 0.0)
Кадр 2:  ██████░░░░░░░░░░░░░░░░░░░░  (+15% расстояния)
Кадр 3:  █████████░░░░░░░░░░░░░░░░░  (+15% остатка)
Кадр 4:  ███████████░░░░░░░░░░░░░░░  (приближается...)
...
Кадр N:  ██████████████████████████  (достигнуто)
```

## Пример расчета в реальном времени

### Входные данные (кадр #1)

```python
# MediaPipe обнаружил руку
landmarks[4] = [0.45, 0.60, -0.05]   # Большой палец
landmarks[8] = [0.50, 0.58, -0.06]   # Указательный палец
landmarks[9] = [0.55, 0.52, -0.08]   # Центр ладони
```

### Шаг 1: Вычисление pinch

```python
thumb_tip = [0.45, 0.60]
index_tip = [0.50, 0.58]

distance = √((0.50 - 0.45)² + (0.58 - 0.60)²)
         = √(0.0025 + 0.0004)
         = √0.0029
         = 0.054

pinch = clip((0.054 - 0.02) / 0.15, 0, 1)
      = clip(0.034 / 0.15, 0, 1)
      = clip(0.227, 0, 1)
      = 0.227  # Пальцы близко друг к другу
```

### Шаг 2: Нормализация позиции

```python
palm_x_raw = 0.55
palm_y_raw = 0.52

normalized_x = (0.55 - 0.5) * 2 = 0.1
normalized_y = (0.5 - 0.52) * 2 = -0.04

# Позиция: немного правее центра, чуть выше центра
```

### Шаг 3: Маппинг к трансформациям

```python
# Позиция
target_position_x = 0.1 * 3.0 = 0.3
target_position_y = -0.04 * 2.0 = -0.08

# Масштаб
target_scale = 0.5 + 0.227 * 1.5 = 0.84

# Вращение (предположим предыдущая позиция была 0.0, 0.0)
velocity_x = 0.1 - 0.0 = 0.1
velocity_y = -0.04 - 0.0 = -0.04

target_rotation_y += 0.1 * 2.0 = +0.2 рад (~11°)
target_rotation_x -= -0.04 * 2.0 = +0.08 рад (~4.5°)
```

### Шаг 4: Интерполяция (предположим текущие значения = 0)

```python
# Lerp factor = 0.15
position_x = 0 + (0.3 - 0) * 0.15 = 0.045
position_y = 0 + (-0.08 - 0) * 0.15 = -0.012
scale = 1.0 + (0.84 - 1.0) * 0.15 = 0.976
rotation_x = 0 + 0.08 * 0.15 = 0.012 рад
rotation_y = 0 + 0.2 * 0.15 = 0.03 рад
```

### Результат (кадр #1)

```
Объект сместился:
  → Вправо на 0.045 единицы
  ↑ Вверх на 0.012 единицы
  ↘ Уменьшился до 97.6% размера
  ⟳ Повернулся на 0.012 рад по X, 0.03 рад по Y
```

### Следующий кадр (#2, через 33мс)

Если рука осталась в той же позиции, интерполяция продолжится:

```python
position_x = 0.045 + (0.3 - 0.045) * 0.15 = 0.083
position_y = -0.012 + (-0.08 + 0.012) * 0.15 = -0.022
scale = 0.976 + (0.84 - 0.976) * 0.15 = 0.956
```

Через ~10 кадров объект достигнет целевой позиции!

## Цветовая схема OpenGL куба

```
        ┌──────────┐
       /  GREEN   /|
      /  (Top)   / |
     └──────────┘  |
     |           |  |
     |  BLUE     | / RED
     |(Front)    |/ (Back)
     |           |
     └──────────┘
    YELLOW
   (Bottom)

Left: CYAN
Right: MAGENTA
```

**RGB значения:**
- Передняя: `(0.3, 0.5, 1.0)` - Синий
- Задняя: `(1.0, 0.3, 0.3)` - Красный
- Верхняя: `(0.3, 1.0, 0.3)` - Зеленый
- Нижняя: `(1.0, 1.0, 0.3)` - Желтый
- Левая: `(0.3, 1.0, 1.0)` - Голубой
- Правая: `(1.0, 0.3, 1.0)` - Пурпурный

## Расширение функциональности

### Добавить новый жест (например, "кулак")

```python
# В HandTracker._calculate_gestures()

# Проверяем расстояние от кончиков пальцев до запястья
wrist = landmarks[0]
fingertips = [
    landmarks[8],   # Index
    landmarks[12],  # Middle
    landmarks[16],  # Ring
    landmarks[20]   # Pinky
]

# Среднее расстояние
avg_dist = mean(||tip - wrist|| for tip in fingertips)

# Если все пальцы близко к запястью → кулак
self.fist_detected = avg_dist < 0.3  # Порог нужно подобрать

# Добавляем в get_gesture_data()
return {
    ...
    'fist_detected': self.fist_detected
}
```

### Добавить новый 3D объект (сфера)

```python
# В GLWidget

def _draw_sphere(self, slices=20, stacks=20):
    """Рисуем сферу через gluSphere"""
    quadric = gluNewQuadric()
    gluSphere(quadric, 1.0, slices, stacks)
    gluDeleteQuadric(quadric)

# В paintGL()
if self.object_type == 'sphere':
    self._draw_sphere()
elif self.object_type == 'cube':
    self._draw_cube()
```

### Добавить частицы при движении

```python
# В GLWidget

class Particle:
    def __init__(self, x, y, z):
        self.position = [x, y, z]
        self.velocity = [random(), random(), random()]
        self.lifetime = 1.0  # секунды

# Создаем частицы при быстром движении
if velocity > threshold:
    particles.append(Particle(position_x, position_y, position_z))

# Обновляем и рисуем частицы
for p in particles:
    p.position += p.velocity * dt
    p.lifetime -= dt
    draw_point(p.position)

particles = [p for p in particles if p.lifetime > 0]
```
