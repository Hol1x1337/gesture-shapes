## 🎮 Управление жестами

### Основные жесты

| Жест | Описание | Действие |
|------|----------|----------|
| 🤏 **Pinch** | Сведите большой и указательный палец, двигайте рукой | Вращение объекта |
| ✌️ **Двойной Pinch** | Обе руки показывают pinch, разводите/сводите руки | Масштабирование (приближение/отдаление) |
| ✋ **Открытая ладонь** | Все пальцы выпрямлены | Остановка всех действий |

### Расширенные жесты ⭐

| Жест | Описание | Действие |
|------|----------|----------|
| 🤛 **Кулак** | Все пальцы согнуты | Пауза вращения |
| ☝️ **Указательный вверх** | Только указательный палец поднят | Сброс позиции объекта |
| ✌️ **Победа (V)** | Указательный и средний пальцы буквой V | Переключение 3D объекта |
| ❤️ **Сердце** | Большой + указательный вместе, остальные выпрямлены | Специальный эффект |

## ⚙️ Технические детали

### Оптимизация производительности

Приложение специально оптимизировано для систем с мощным CPU и встроенной GPU:

- **MediaPipe Hands → CPU**: Принудительное использование процессора для распознавания жестов
- **OpenGL → GPU**: Аппаратное ускорение 3D рендеринга
- **Многопоточность**: Параллельная обработка на 4 ядрах CPU
- **Результат**: FPS 45-60 при загрузке CPU 70-85%

Подробнее: [CPU_GPU_OPTIMIZATION.md](CPU_GPU_OPTIMIZATION.md)

### Системные требования

**Минимальные:**
- CPU: 2 ядра, 2.0 GHz
- RAM: 4 GB
- GPU: Любая (встроенная подходит)
- Камера: 720p

**Рекомендуемые:**
- CPU: 4+ ядра, 3.0+ GHz
- RAM: 8 GB
- GPU: Дискретная (опционально)
- Камера: 1080p

## 🐛 Решение проблем

### Низкий FPS (<30)

1. Закройте другие приложения
2. Уменьшите `cpu_threads` в настройках
3. Измените `render_quality` на "medium"

### Плохое распознавание жестов

1. Улучшите освещение
2. Держите руку на расстоянии 30-60 см от камеры
3. Используйте фон без лишних деталей

### Ошибки запуска

1. Проверьте установку зависимостей: `pip install -r requirements.txt`
2. Убедитесь, что камера подключена
3. Проверьте логи в папке `logs/`
